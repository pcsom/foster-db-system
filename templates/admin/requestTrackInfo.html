About the Request Tracking System

Each request made by a family is assigned a status indicator:
A request labelled with (P) means that it is pending (the family submitted it recently, and item collection hasn't started yet).
(C) means that the items for the request are currently being collected by NAFC.
(R) means that the items for the request are ready to be delivered/picked up.
(F) denotes that the family has received the items and the request has been fulfilled.

A request is set to (P) immediately after the family submits it.
A request is set from (P) to (C) when you generate a 'requests printout' for the respective family. (do it HERE)
A request is set from (C) to (R) when you send the respective family an alert that their items are ready for pickup. (do it HERE)
A request is set from (R) to (F) when you make a distribution log entry for the respective family. (do it HERE)
